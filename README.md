ein paar Regeln:
1. Es soll Spaß machen!
2. Es geht darum zu lernen.
3. Hab' keine Angst Fragen zu stellen.
4. Hab' keine Angst Beifahrerin zu sein.
5. Wir wollen positiv auseinander gehen.
6. Halte dich an die Rollenverteilung.
7. Sei nicht unfreundlich.
8. Beharre nicht auf deiner Lösung 
   -Hast du einen Grund für dieses Vorgehen?
